# 照片GPS信息写入工具

一个功能完整的图形化应用程序，用于从GPX或FIT轨迹文件中提取GPS数据，并批量写入照片的EXIF元数据中。
本软件用ai写成，很好的解决了我相机拍照照片没有gps数据的问题。如果有大佬对此也感兴趣可以交流交流，如果能把这个丑陋的ui优化一下就更好了，ai写的ui简直太难看了

## 功能特性

### 1. 图形用户界面
- 直观易用的操作界面
- 支持文件拖放导入
- 实时进度显示
- 详细的操作日志

### 2. GPS数据处理
- 支持GPX和FIT格式的轨迹文件
- 基于时间戳的精确匹配算法
- 准确写入GPS坐标到EXIF

### 3. GPX文件管理
- 自动根据时间范围重命名GPX文件
- 命名格式: YYYYMMDD_HHMMSS_YYYYMMDD_HHMMSS.gpx

### 4. 照片格式支持
- 常见格式: JPEG, PNG, TIFF
- RAW格式: CR2, NEF, ARW, DNG等
- 不修改非GPS相关的EXIF信息

### 5. 其他功能
- 时间偏移调整（处理时区问题）
- 最大时间差设置
- 日志保存功能
- 批量处理支持

## 安装和运行

### 方法一: 直接运行Python脚本（需要Python环境）

1. 确保已安装Python 3.8或更高版本
2. 安装依赖包:
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序:
   ```bash
   python main.py
   ```
   或双击 `run.bat`

### 方法二: 打包为独立可执行文件

1. 运行打包脚本:
   ```bash
   build.bat
   ```
2. 打包完成后，可执行文件位于 `dist/PhotoGPSWriter.exe`
3. 直接双击运行即可，无需Python环境

## 使用说明

### 基本操作流程

1. **选择GPS轨迹文件**
   - 点击"选择文件"按钮，选择GPX或FIT文件
   - 或直接拖放文件到窗口中
   - 可点击"重命名GPX"按钮自动重命名GPX文件

2. **选择照片文件**
   - 点击"选择文件"选择单个或多个照片
   - 点击"选择文件夹"扫描整个文件夹
   - 或直接拖放照片/文件夹到窗口中

3. **调整设置（可选）**
   - 时间偏移: 调整照片时间与GPS时间的时差（小时）
   - 最大时间差: 设置照片与GPS点的最大允许时间差（秒）

4. **开始处理**
   - 点击"开始处理"按钮
   - 查看实时日志和进度条
   - 处理完成后查看统计信息

5. **保存日志（可选）**
   - 点击"保存日志"按钮保存处理记录

## 注意事项

1. **备份照片**: 建议在处理前先备份原始照片
2. **时间同步**: 确保相机时间与GPS设备时间尽可能一致
3. **时区问题**: 如果相机和GPS设备时区不同，使用时间偏移功能调整
4. **RAW格式**: 部分RAW格式可能需要特殊软件查看写入的GPS信息

## 项目结构

```
├── gps_parser.py        # GPS文件解析模块

├── gps_matcher.py       # GPS匹配算法
├── requirements.txt     # 依赖包列表
├── build.bat           # 打包脚本
├── run.bat             # 运行脚本
└── README.md           # 说明文档
```

## 依赖库

- fitparse: FIT文件解析
- Pillow: 图像处理
- piexif: EXIF数据读写
- pyinstaller: 打包工具
